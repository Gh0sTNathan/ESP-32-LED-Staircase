# ESP-32-LED-Staircase
ESP32-controlled LED staircase with 2 PIR motion sensors


This is a repo for my LED staircase for my apartment, including 2 PIR sensors for motion activation and a web server for easy control access, ran off an ESP32 microcontroller. 
Wiring, soldering, and installation done by me. 

AI was used for the overall code, integration by myself.
I have switched to WLED due to its vast functionality and stability.
I also do not have the time to carry on with this project, as I want to move on to other home IOT projects for my apartment

## List of materials and devices used: 
- ~100m of 12AWG wiring
- 2x HC-SR501 PIR Motion sensors
- 1x ESP32 Dev Board
- 1x LRS Compact 5V 18A 100W PSU
- 300 WS2812B LED's
